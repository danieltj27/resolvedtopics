# Resolved Topics

Set a topic as solved by selecting a post as a resolution.

## Features

- Allows users of a forum to mark topics as resolved by selecting a post that resolved it.
- Adds two new permissions, one to allow topic starters to mark their topics as resolved and one to allow moderators to mark all topics as resolved regardless of whether they are the topic starter.

## Install

1. Add the extension files into the `ext/danieltj/resolvedtopics` directory.
2. Log in to the Admin Control Panel and install the extension.
3. Adjust your forum based permissions to decide who can mark topics as resolved.
